# LLM Text Analysis

A Python project for text analysis using Large Language Models (LLMs) with a Streamlit frontend, FastAPI backend, and pre-trained HuggingFace models.

---

## Setup

Clone the repo:

```bash
git clone https://github.com/Annyalvareznogales/llm-text-analysis.git
cd llm-text-analysis
```

### Development

```bash
docker compose up
```
